# RAKSHA SAFETY - EXTREME RISK AI
## BIHAR FIRST. BIHARI FIRST. INDIA FIRST.

**Voice First AI for 108 Emergency**  
**Made in Patna, Bihar 🇮🇳**

### PROBLEM
Bihar ke gaon aur raat me mahilao ko turant help nahi milti. Police tak location pahunchane me der ho jati hai.

### SOLUTION  
1 Click SOS + Live Location + ML Risk Prediction  
`EXTREME RISK` detect hote hi turant alert.

### TECH STACK
`Python` | `Flask` | `TensorFlow` | `Vercel` | `AI/ML`
